Basics Of HTML :

1) Basic HTML Structure :
     - Doctype Declaration:
          <!Doctype html>
     - Structure
          <html>
               <head>
                    <title> Title Name </title>
               </head>

               <body>
                    Main Content Goes here
               </body>
          </html>
2) Common Tags :
     - headings : h1 to h6
     - paragraph : p
     - Lists :
          - Ordered : <ol> <li> </li> </ol>
          - Un-Ordered : <ul> <li> </li> </ul>
     - Links : <a url="" href="">
     - Image : <img src="" alt="">
     -buttons : <button> </button>
3) Semantic Tags :
     <li>header</li>
